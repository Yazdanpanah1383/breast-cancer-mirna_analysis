# Breast Cancer miRNA Analysis

This project investigates serum miRNA expression patterns in breast cancer using GEO dataset GSE73002.

## Dataset
- GEO: GSE73002
- Sample type: Serum
- Groups:
  - Breast cancer
  - Benign breast disease
  - Non-cancer controls

## Analyzed miRNAs
- miR-29a-3p
- let-7a-5p
- miR-16-5p
- miR-17-5p

## Methods
- GEO dataset analysis
- Expression comparison
- Boxplot visualization using Excel

## Key Findings
- miR-17-5p showed elevated expression in cancer samples
- let-7a-5p showed lower expression patterns
- Overlap existed between benign and cancer groups
